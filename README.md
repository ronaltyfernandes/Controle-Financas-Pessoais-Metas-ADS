# Controle Financeiro

Sistema de controle financeiro desenvolvido em Python para gerenciamento de categorias, entradas e saídas financeiras.

## Funcionalidades

- CRUD de categorias
- CRUD de entradas
- CRUD de saídas
- Listagem de registros
- Controle básico de finanças pelo terminal

## Requisitos

- Python 3.10 ou superior
- Pip

## Instalação

Clone o repositório:

```bash
git clone <https://github.com/ronaltyfernandes/Controle-Financas-Pessoais-Metas-ADS.git>
cd <Controle-Financas-Pessoais-Metas-ADS>
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

## Executando o projeto

Execute o arquivo principal da aplicação:

```bash
python main.py
```

Ou, dependendo da sua versão do Python:

```bash
python3 main.py
```
